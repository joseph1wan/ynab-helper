# ynab-helper

Match Target order line items to uncategorized YNAB transactions and propose split categorizations.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
playwright install chromium
cp .env.example .env
# Add YNAB_TOKEN to .env
```

## Usage

```bash
# One-time: sync YNAB categories
ynab-helper sync-categories

# One-time: log in to Target (manual MFA in browser)
ynab-helper target-login

# Fetch orders, match, and write proposals
ynab-helper fetch

# Review and approve in browser
ynab-helper review

# Revert last approved split
ynab-helper undo
```

## Configuration

- `config/config.yaml` — YNAB token, budget ID, payee pattern
- `config/categories.json` — generated by `sync-categories`
- `config/rules.yaml` — keyword regex → category mappings

## Cursor skill

For unmatched Target items, use the `categorize-unmatched` skill to suggest categories and new regex rules.
